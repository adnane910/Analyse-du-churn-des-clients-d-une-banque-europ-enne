# 🏦 Analyse du churn des clients bancaires

## 📌 Description
Ce projet a pour objectif d’analyser et de prédire le désabonnement (churn) des clients d’une banque européenne.  
Il s’appuie sur un jeu de données de 10 000 clients contenant des informations démographiques, financières et comportementales.

## 🎯 Objectifs
- Identifier les facteurs influençant le churn  
- Analyser le profil des clients  
- Comparer les comportements selon le pays  
- Segmenter les clients  
- Prédire le churn à l’aide de modèles de machine learning  

## 📊 Contenu du projet
- Analyse exploratoire des données (EDA)  
- Identification des facteurs clés du churn  
- Segmentation des clients  
- Modélisation prédictive  
- Dashboard interactif avec Streamlit  

## 🛠️ Technologies utilisées
- Python  
- Pandas  
- Matplotlib / Seaborn  
- Scikit-learn  
- Streamlit
